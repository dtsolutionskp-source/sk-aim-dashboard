# SK AIM 데모 대시보드

공공기관(지자체, 관광공사, 축제 운영기관) 대상 SK AIM 서비스 영업·제안용 데모 대시보드입니다.

**해오름 야간문화축제** 가상 데이터를 활용합니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 메뉴 구성

1. **홈** - 플랫폼 소개 및 핵심 메시지
2. **마케팅** - 홍보 KPI, 타겟 설정, 소재 관리, 지역별 반응
3. **데이터 분석** - 방문객 특성, 유입 지역, 체류시간, 공간 Grid 분석
4. **설문 분석** - 만족도, NPS, 소비 분석, AI 의견 요약
5. **성과 리포트** - 자동 생성 보고서 및 다운로드

## 기술 스택

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router
- Lucide Icons
